# ClientPulse — Name Suggestions

Brainstorm list of alternative product names. Current name **ClientPulse** is strong; this doc captures alternatives explored if rebrand is considered.

**Last updated:** 2026-05-12.

---

## Top 3 picks

| Name | Why |
|------|-----|
| **ClientPulse** | Already brand. Clear "pulse on client work" metaphor. Domain `clientpulse.dev` in use. |
| **LoopBrief** | Matches repo dir `brief_loop`. Dual meaning — *brief* (update) + *loop* (close feedback). |
| **Cadence** | Premium, short, calm. Fits agency-client rhythm of recurring updates. |

## Rejected / weak

- **BriefBot** — implies AI/automation, not core feature
- **Standup** — too engineering-coded, alienates non-dev agencies
- **Catchup** — connotes lagging, opposite of brand promise
